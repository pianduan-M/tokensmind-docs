> OpenClaw is an open agent platform that runs on your machine and works through chat apps you already use.

## Install

<Note>
  Use Node **≥ 22** before installing.
</Note>

```shellscript theme={null}
npm install -g openclaw@latest
```

### Terminal onboarding

```shellscript theme={null}
openclaw onboard
```
