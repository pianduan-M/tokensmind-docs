# API group types

## Groups and characteristics

| Group | Type | Rate | Supported models |
| :-- | :-- | :-- | :-- |
| default | Default; official-style routing | Official rate ×1 | OpenAI, Claude, domestic models |
| Claude code | Claude-only; not for general chat | Official rate ×1 | Claude (code workflows) |
| Codex | ChatGPT-only; not for general chat | Official rate ×1 | OpenAI |
| Official native | Direct vendor routing; stable for enterprise | Official rate ×1 | OpenAI, Claude, domestic models |
| Limited promo | Pooled accounts; cost-effective for individuals | Official rate ×1 | OpenAI, Claude, domestic models |

> Set the group per token on the **Tokens** page.
