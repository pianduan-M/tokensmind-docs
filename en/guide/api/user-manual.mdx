The API sandbox lets you send live requests and inspect responses.

<Info>
  For safer code:

- Keep secrets in environment variables (e.g. in Python use `os.getenv("TOKENSMIND_API_KEY")`).
- Do not log or print API keys.
- Do not commit keys—add `.env` to `.gitignore`.
  </Info>
