在我们的系统中，“分组”具有特殊意义，主要体现在渠道、价格与选择空间三个方面。

## 1. 渠道来源差异

- 同一模型（如 GPT、Claude 等）在不同分组中可能对应不同的渠道来源
- 渠道来源不同，会直接带来各分组之间的成本差异，从而形成不同的定价

## 2. 价格多样性

- 市面上 API 价格不统一，根本原因之一在于渠道与成本结构存在差异
- 我们采用**明码实价**策略：价格与规则清晰展示，确保透明可查

## 3. 用户自主选择

- 用户可根据性能、稳定性、价格等因素选择最适合的分组
- 为不同预算与场景提供更灵活、个性化的配置空间

# 官方费率说明

为帮助用户直观理解各分组价格与官网原价的关系，我们引入“**官方费率**”概念：

- 本站以**人民币**计费
- 模型页展示的价格为**官网原价（折算为人民币）**
- 各分组最终计费价格 = **官网原价 × 分组费率**

## 费率示例

1. **分组费率 = 1**
   - 按官网原价计费（与官方一致）

2. **分组费率 = 0.4**
   - 按官网原价的 **0.4 倍**计费（即 **4 折**）

通过“官方费率”，用户可以更清晰地对比不同分组的价格差异，并根据需求与预算做出更合适的选择。
